# Loyal Spawn Plugin

Simple configurable spawn plugin for Paper / Spigot servers.

## Features

- /setspawn command
- /spawn teleport
- configurable teleport delay
- teleport cancel if player moves
- configurable console command
- cooldown system
- config reload command
- title + sound effect

## Commands

/setspawn
/spawn
/spawn reload

## Compatibility

Tested on:

Paper 1.21.1
Spigot
Purpur
